# Gestor de Inventario - Swing

Este proyecto es un **Gestor de Inventario** desarrollado en **Java** con **interfaz gráfica Swing**. Permite agregar, mostrar, buscar, editar y eliminar productos de manera visual, usando formularios y ventanas emergentes.

 Proyecto de práctica para mejorar habilidades en Java y Swing.


---

## Funcionalidades

- Agregar productos con nombre, precio, cantidad y descripción.
- Mostrar todos los productos registrados.
- Buscar un producto por nombre (case-insensitive).
- Editar un producto existente.
- Eliminar productos del inventario.

---

## Tecnologías usadas

- Java SE
- Swing (Interfaz gráfica)
- ArrayList para manejar múltiples productos
- JOptionPane para interacción con el usuario

## Cómo ejecutar el proyecto

Clonar el repositorio: git clone https://github.com/bryanmotta1296/InventarioSwing.git

Abrir el proyecto en IntelliJ IDEA.

Ejecutar la clase InterfazInventario.java.

Usar la interfaz para agregar, buscar, editar o eliminar productos.
